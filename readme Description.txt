welcome,
Online school web app built with asp.net MVC and access token to identify its users,

this app has two users which are Instructor and Student 

Instructor upload course and is divided into segments and it could be text or video link 
and finally, there is a test to make on each particular part of this course.

the student can take this course as a registered user and of course with its tests and store the result 
and there are the anonymous user who just takes the course or download sheet paper or course material

and the course page which the students can search for a specific course or by category and the latest added courses as well.
