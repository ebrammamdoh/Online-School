welcome,

Online school web app built with asp.net MVC and access token to identify its users,

this app has two users which is Instructor and Student 

Instructor upload course and it divided into segments and it could be text or video link 
and finally there is a test to make on each particular part of this course.

student can take this course as a regestered user and ofcourse with its tests and store the result 
and there are anonymous user who just take the course or download sheet paper or course material

and the course page which the students can search for specific course or by category and the latest added courses as well.
